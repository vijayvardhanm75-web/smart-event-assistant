# smart-event-assistant

## Problem
Large-scale events such as stadiums face challenges like crowd congestion, long queues, and poor navigation.

## Solution
This project proposes an AI-powered Smart Event Assistant that improves attendee experience using real-time insights and intelligent decision-making.

## Features
- Smart crowd prediction
- Gate recommendation system
- Queue time estimation
- Real-time alerts

## Technology Stack
- AI Prompt Engineering
- Python (Decision Logic)
- IoT (conceptual)
- Mobile App Interface (concept)

## How it Works
The system collects crowd data and uses AI-based prompts and logic to guide users toward less crowded areas and reduce waiting time.

## Repository Contents
- prompt.txt → AI prompt used
- logic.py → decision-making logic
- sample_output.txt → generated output

## Impact
- Better crowd management
- Reduced waiting times
- Improved user experience
