Problem:
Large-scale events face issues like overcrowding, long waiting times, and lack of real-time coordination.

Solution:
An AI-powered Smart Event Assistant that guides attendees using real-time data.

Features:
- Crowd heatmap visualization
- Smart gate recommendation
- Real-time alerts
- Queue time prediction

Technology Stack:
- AI (Prompt Engineering)
- Python (Decision Logic)
- IoT Sensors
- Mobile Application

Impact:
- Reduced waiting time
- Improved crowd management
- Enhanced attendee experience
